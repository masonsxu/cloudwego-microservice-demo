  fast-check:
    runs-on: ubuntu-latest

      - name: Run converter tests

      - name: Run validator tests
        run: go test ./biz/logic/*/validation_test.go -v

#### 阶段2：完整集成测试（合并到main后）
```yaml
# .github/workflows/integration-test.yml
name: Integration Test
on:
  push:
    branches: [main]
jobs:
  integration-test:
    runs-on: ubuntu-latest
    steps:
      - name: Setup podman
        run: |
          sudo apt-get install podman
          sudo systemctl start podman
      
      - name: Run all tests with testcontainers
        env:
          TESTCONTAINERS_DOCKER_SOCKET: unix:///run/podman/podman.sock
        run: go test ./... -v
```
